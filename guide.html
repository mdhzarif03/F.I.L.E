File Intelligence and Local Engine (F.I.L.E.)

Follow these instructions to set up and run the application safely on Windows.

1. Requirements

  • Windows Operating System
  • Python 3.x installed (with Tkinter included by default)

2. Setup Instructions

  1. Save your Python script exactly as organiser.py inside your preferred working directory.
  2. If you have an icon file, place your .ico file in the same folder.

3. How to Run the Application

  1. Open Command Prompt or PowerShell.
  2. Navigate to the folder where your script is stored using the cd command:
    cd "D:\F.I.L.E"
  3. Launch the program by running:
    python organiser.py

4. How to Use F.I.L.E.

  1. Click the Browse... button on the application interface.
  2. Select the directory you wish to clear up (e.g., your Downloads folder).
  3. Click Organize Folder to execute the sorting algorithm.
Security Check: This application relies strictly on standard Windows environment libraries (os, shutil, tkinter). It does not require internet access, third-party registry changes, or elevated administrative privileges to operate.
